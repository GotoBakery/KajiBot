# メッセージ設定機能 (Config Sheet)

`config` シートを作ることで、ポイント差（Gap）に応じた労いメッセージと色を自由にカスタマイズできます。

## 1. シートの設定
スプレッドシートに `config` というシートを追加し、以下のカラムを作ってください。
**1行目はヘッダーです。**

| A列 (閾値) | B列 (メッセージ) | C列 (色コード) |
|---|---|---|
| 0 | 🕊️ 平和です。仲良くしましょう。 | #57F287 |
| 100 | 🍰 差が開いてきました。スイーツを買いましょう。 | #F1C40F |
| 500 | 🚨 限界突破！休日の献上が必要です。 | #ED4245 |

- **閾値**: 現在のポイント差がこの数値以上になったら、そのメッセージが採用されます（一番高い閾値が優先されます）。
- **メッセージ**: Discordのパネルに表示される文章です。
- **色コード**: `#RRGGBB` 形式で指定してください。
    - 緑系: `#57F287`
    - 黄色系: `#F1C40F`
    - オレンジ系: `#E67E22`
    - 赤系: `#ED4245`

## 2. 反映
設定後、Discordで「🔄 更新」ボタンを押すと反映されます。
（`config` シートがない、または空っぽの場合は、デフォルトのメッセージが表示されます）
